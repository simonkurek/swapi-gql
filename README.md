## Description

SWAPI-GQL is a GraphQL wrapper for the Star Wars API (SWAPI). It is built using NestJS and Apollo Server.

## Architecture Design

### Description

App has 3 layers.
Each layer is a separate module.
API is handling all GQL staff.

### API

Api is built using GQL and Apollo Server.

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
